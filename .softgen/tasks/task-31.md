## Notes
Interface admin complète pour gérer les biens affichés sur le site public : création, modification, suppression et publication/dépublication.

## Checklist
- [ ] Page admin dédiée aux biens publics
- [ ] Liste tous les biens (publiés et non publiés)
- [ ] Badge visuel statut publication
- [ ] Bouton basculer publication (publier/dépublier)
- [ ] Bouton créer nouveau bien
- [ ] Formulaire complet création bien (référence, titre, type, prix, localisation, etc.)
- [ ] Upload photos multiples
- [ ] Upload vidéo (optionnel)
- [ ] Bouton modifier bien existant
- [ ] Bouton supprimer bien
- [ ] Confirmation avant suppression
- [ ] Filtres par statut publication
- [ ] Recherche rapide

## Acceptance
- L'admin peut créer, modifier, supprimer des biens
- L'admin peut publier/dépublier un bien en un clic
- Seuls les biens publiés apparaissent sur le catalogue public
- L'interface est intuitive et responsive
