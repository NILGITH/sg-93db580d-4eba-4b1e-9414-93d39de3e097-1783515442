## 🔧 Dépannage

**Problème : "Failed to fetch"**
→ Vérifiez que le serveur Next.js est démarré
→ Ouvrez la console du navigateur (F12) pour voir les logs détaillés
→ Redémarrez le serveur si nécessaire

**Problème : "Invalid credentials"**
→ Vérifiez l'email et le mot de passe (respectez les majuscules)
→ Utilisez les comptes listés ci-dessus (déjà créés)

**Problème : "Profil utilisateur introuvable"**
→ Les profils sont créés automatiquement avec les comptes
→ Contactez le support technique si le problème persiste
